# Noverra ToDo (Frontend-only)

Minimal Next.js + Tailwind frontend app (App Router) â€” stores tasks in localStorage.
Designed for quick offline usage as a simple task board.

## Quick start

1. Install dependencies:
   ```
   npm install
   ```

2. Run dev server:
   ```
   npm run dev
   ```

3. Open http://localhost:3000

## Build to production
```
npm run build
npm start
```

## Notes
- App uses localStorage key `noverra_tasks_v1`
- Dark mode is default (set via `class="dark"` in layout)
